# 🌍 Automated Weather Data Pipeline

A fully automated data pipeline that fetches real-time weather data 
from 5 cities, stores it in a cloud database, and exports it to Google Sheets.

## 🔧 Tech Stack
- **Python** — data fetching & transformation
- **Pandas** — data cleaning & manipulation
- **Open-Meteo API** — free real-time weather data (no API key needed)
- **Turso** — cloud SQL database
- **Google Sheets** — automated reporting
- **Google Colab** — cloud execution environment

## 🔄 Pipeline Flow
Open-Meteo API → Python/andas → Turso Database → Google Sheets
## 🌆 Cities Tracked
- Algiers, Algeria
- Oran, Algeria
- London, UK
- Paris, France
- New York, USA

1. Open the notebook in Google Colab
2. Run all cells in order
3. Data is automatically fetched, stored and exported
